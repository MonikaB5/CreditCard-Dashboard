# Credit Card Usage Analysis Dashboard

This repository contains my MCA project developed using Microsoft Power BI.  
It integrates demographic, financial, and transactional data to provide interactive dashboards, revenue insights, customer segmentation, and risk evaluation.  

## Contents
- Power BI Project File (.pbix)
- SQL Queries
- DAX Formulas
- Screenshots of Dashboard
- Project Report (PDF/DOCX)

## How to Use
Download the `.pbix` file and open it in Power BI Desktop.  
Screenshots are available in the `/screenshots` folder for quick reference.
